# User Settings

**Version:** 1.8.2

## My Account

Personal settings for current user.

## Profile
- Display name
- Email address
- Language preference
- Theme selection (8 themes available)

## Security

### Change Password
- Current password
- New password (min 12 chars)
- Confirm new password

### WebAuthn / FIDO2
- Register security key
- Manage registered credentials
- Remove old keys

See [WebAuthn-Support](WebAuthn-Support) for details.

### API Tokens
- Generate new token
- View active tokens
- Revoke tokens
- Set expiration

### mTLS Certificates
- Upload client certificate
- View mapped certificates
- Remove certificate mapping

See [MTLS-Authentication](MTLS-Authentication) for details.

## Notifications

Subscribe to:
- ☑ Certificate expiration warnings
- ☑ Revocation notices
- ☑ SCEP request approvals
- ☑ System alerts

## Activity Log

View your recent activity:
- Certificates created
- Revocations performed
- Logins
- API usage
