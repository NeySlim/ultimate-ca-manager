# Dashboard

**Version:** 1.8.2

## Overview

The Dashboard provides at-a-glance statistics and quick access to common tasks.

## Widgets

### Certificate Statistics
- Total certificates issued
- Active certificates
- Expired certificates  
- Expiring soon (30 days)

### CA Statistics
- Number of CAs
- Root CAs
- Intermediate CAs
- Sub-CAs

### Recent Activity
- Latest certificates issued
- Recent revocations
- Pending SCEP requests
- ACME orders

### OCSP/CRL Stats
- OCSP requests (24h)
- CRL updates
- Revocation count

## Quick Actions
- Create Certificate
- Create CA
- Import Certificate
- View Pending Requests

See [User-Manual](User-Manual) for detailed usage.
